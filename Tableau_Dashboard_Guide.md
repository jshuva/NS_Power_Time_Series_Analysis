# MCDA 5580: Tableau Dashboard Development Guide

This guide details exactly how to construct the interactive Tableau dashboard for Task 3 using the exported CSV datasets from our Jupyter Notebook.

## 1. Datasets Used
Connect Tableau to the following files generated by `Assignment.ipynb`:
1. **`tableau_historical_data.csv`**: Contains all 2015-2024 hourly records, regions, anomaly flags, and weather data.
2. **`tableau_forecast_data.csv`**: Contains the 7-day Halifax test data comparing `actual_consumption`, `prophet_forecast`, and `lstm_forecast`.

---

## 2. Dashboard Visualizations (The 8 Required Charts)

### 1. Time Series Line Chart of Consumption Over Time
*   **Data Source**: `tableau_historical_data.csv`
*   **Columns**: `timestamp` (Continuous, Exact Date/Time or Day) on **Columns**, `consumption_kwh` (Average or Sum) on **Rows**.
*   **Filters**: Add `region` and `timestamp` (Years) to the filter shelf; show filter controls on the dashboard.
*   **Insight Caption**: "Consumption shows strong daily cyclicality and seasonal spikes."

### 2. Heatmap: Average Consumption by Hour of Day and Day of Week
*   **Data Source**: `tableau_historical_data.csv`
*   **Columns**: `hour` (Discrete Dimension) on **Columns**.
*   **Rows**: `day_of_week` (Discrete Dimension) on **Rows**.
*   **Marks**: `consumption_kwh` (Average) on **Color** (Square mark type). 
*   **Insight Caption**: "Peak energy usage clusters heavily between 4 PM and 8 PM on weekdays."

### 3. Seasonal Consumption Comparison Across Regions
*   **Data Source**: `tableau_historical_data.csv`
*   **Columns**: `region` on **Columns**.
*   **Rows**: `consumption_kwh` (Average) on **Rows**.
*   **Color/Group**: Drop `season` onto the **Color** shelf to create grouped bars or a stacked bar chart.
*   **Insight Caption**: "Winter drives significantly higher electricity demand across all five regions."

### 4. Anomaly Timeline
*   **Data Source**: `tableau_historical_data.csv`
*   **Columns**: `timestamp` (Continuous, Day) on **Columns**.
*   **Rows**: `consumption_kwh` on **Rows**.
*   **Marks**: Change Mark Type to Line. Create a Dual Axis or use Color to highlight anomalies. Drop the `algo_anomaly` (or true `anomaly_flag`) dimension onto the **Color** or **Size** shelf. 
*   **Insight Caption**: "Isolation Forest detects sudden demand spikes and sustained drops during extreme weather."

### 5. Renewable Energy % Trend (2015–2024)
*   **Data Source**: `tableau_historical_data.csv`
*   **Columns**: `timestamp` (Continuous, Year/Month) on **Columns**.
*   **Rows**: `renewable_pct` (Average) on **Rows**.
*   **Insight Caption**: "The grid shows steady long-term growth in the percentage of renewable energy sources."

### 6. Forecast vs. Actual Comparison Chart
*   **Data Source**: `tableau_forecast_data.csv`
*   **Columns**: `timestamp` (Exact Date) on **Columns**.
*   **Rows**: Measure Values on **Rows**. 
*   **Filters**: Filter Measure Names to include only `actual_consumption` and `prophet_forecast` (or LSTM).
*   **Marks**: Color by Measure Names (Line chart).
*   **Insight Caption**: "The forecasting model accurately captures the daily demand valleys and peaks over the final 7 days."

### 7. Regional Consumption Breakdown (Map or Bar)
*   **Data Source**: `tableau_historical_data.csv`
*   **Map Setup**: Double click `region` (ensure Tableau recognizes it as a geographic location, manual assignment required in NS) or build a simple Bar Chart.
*   **Marks**: `consumption_kwh` (Sum or Average) on Size/Color.
*   **Insight Caption**: "Halifax dominates average electricity load compared to rural counties."

### 8. CO₂ Emissions Trend Over Time
*   **Data Source**: `tableau_historical_data.csv`
*   **Columns**: `timestamp` (Continuous, Month/Year) on **Columns**.
*   **Rows**: `co2_kg` (Sum) on **Rows**.
*   **Insight Caption**: "Overall CO₂ emission volume trends downward as renewable grid integration increases."

## 3. Dashboard Assembly & Interactivity

1. **New Dashboard**: Create a new dashboard pane, sizing it to at least generic Desktop (1000 x 800) for a clean layout.
2. **Layout**: Drag and drop the 8 generated sheets onto the dashboard. Use layout containers to align them cleanly (e.g., KPIs/Timelines up top, regional breakdowns below).
3. **Interactivity**:
   *   Hover over the **Region filter** (from Chart 1) -> Click the dropdown -> **Apply to Worksheets > All Using This Data Source**. 
   *   Do the same for the **Year filter**.
   *   This fulfills the "2 interactive filters" requirement.
4. **Final Polish**:
   *   Add a large text box at the top for the title: **"Nova Scotia Power Electricity Analytics"**.
   *   Ensure every chart has a clear, readable title.
   *   Verify the one-sentence insight captions are visible directly underneath each chart.
   *   File -> Export as Packaged Workbook (`.twbx`) for submission.
